Florida Scratch-Off Statistician

Not affiliated with, endorsed by, or sponsored by the Florida Lottery or the State of Florida. An independent hobby project.

Data: the Florida Lottery's public game API + public Top-Prize Winner reports (see last queried per game). Prizes and odds change constantly — verify with the Florida Lottery before purchasing. Expected-value figures use advertised face values; most jackpot winners take the cash option at roughly 60–65% of the advertised prize. Every scratch-off is negative expected value as designed (a nearly sold-out game can briefly turn positive on remaining tickets — we flag those) and every ticket is independent — nothing here changes your odds.
